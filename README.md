# Buscador de GIFs
 - Este app utiliza uma [API](https://developers.giphy.com/) de GIFs.
 - Os GIFs são mostrados em uma grade.
 - Eles podem ser pesquisados. Retornará os 19 primeiros GIFs e poderá ser carregado os próximos 19 caso aperte no botão no final da página.
 - Os GIFs de *trending* são mostrados inicialmente quando abre o app ou se a pesquisa for vazia.
 - O GIF poderá ser compartilhado caso fique precionando-o, ou
 - Ao clicar no GIF, será direcionado para uma página com a imagem do GIF, e terá, na appBar, uma opção de compartilhar.
